MaskLinks
Программа на Go, которая находит в тексте ссылки, начинающиеся с http://, и заменяет их адрес (все символы после http:// до пробела) на звёздочки *

Установка
Убедитесь, что установлен Go версии 1.21 или выше.

Склонируйте репозиторий и перейдите в папку:

git clone https://github.com/qwaseri832/masklinks.git
cd masklinks

Использование
Запуск программы:

go run main.go

Пример
Входная строка:
Hello, its my page: http://localhost123.com See you

Выходная строка:
Hello, its my page: http://************* See you

Как это работает
Программа ищет подстроку http://

Когда находит, копирует http:// в результат без изменений

Все следующие символы до первого пробела заменяет на *

Остальной текст копирует как есть

Ограничения
Поддерживаются только ссылки, начинающиеся с http:// (не https://)

Ссылка должна заканчиваться пробелом или концом строки

Не обрабатываются ссылки в кавычках или скобках
